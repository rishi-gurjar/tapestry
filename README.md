### Tapestry: expand your knowledge base, derived from your current interests

###### HOW TO SETUP?
```
> * cd api
> touch .env
```
Add your OpenAI key as "OPENAI_API_KEY=<enter_key_here>" in the .env file

###### HOW TO RUN (from root)?
```
> npm i
> npm run dev
```
simultaneously in another terminal, run
```
> * cd api
> flask run
```
